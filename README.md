# Process-modeling
Данный проект выполнен в рамках анализа бизнес-процессов для сети медицинских клиник «Вита». Целью было описать текущий (as-is) процесс приёма у врача, выявить узкие места и предложить целевую (to-be) модель с использованием мобильного приложения для автоматизации записи пациентов.
